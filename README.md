# CalorieAI — Calories Tracker 🥗

A beautiful, mobile-first calories tracking web app with dark mode, food scanning, and offline support.

## Features
- 📊 **Food Diary** — Track daily calories with a visual progress ring
- 🔍 **Food Scanner** — Identify ingredients with calorie overlay labels
- ⚙️ **Settings** — Customize gender, height, weight, goal & activity level
- 🌙 **Dark Mode** — Beautiful dark theme with smooth toggle
- 📱 **PWA** — Install on your phone's home screen, works offline

## How to Use

### Run Locally
Just open `index.html` in any modern browser.

### Deploy to GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages → Source: main branch**
3. Your app will be live at `https://yourusername.github.io/repo-name/`

### Install on Phone
1. Open the deployed URL on your phone
2. **Android**: Tap the "Add to Home Screen" banner (or Menu → Add to Home Screen)
3. **iPhone**: Tap Share → "Add to Home Screen"
4. The app opens fullscreen like a native app!

## Tech Stack
- Vanilla HTML, CSS, JavaScript (no frameworks)
- LocalStorage for data persistence
- Progressive Web App (manifest + service worker)
- Google Fonts (Inter)
